## Objective

### TypeScript Definition (protocol.ts:266-272)
```typescript
export interface FileWrite {
  type: 'FileWrite';
  worktreeId: string;
  path: string;

  encoding?: 'utf8' | 'base64';
}
```

### Unified Schema
- `worktreeId: string` (Uuid serialized as string)
- `path: string`
- `content: string`
- Note: TypeScript has optional `encoding?: 'utf8' | 'base64'` field not present in Rust
