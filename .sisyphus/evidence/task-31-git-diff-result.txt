# Task 31: GitDiffResult Schema Discovery and Fixture Creation

## Schema Synthesis

### Rust Definition (crates/ws-server/src/protocol.rs)
```rust
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, ts_rs::TS)]
#[serde(rename_all = "camelCase")]
#[ts(export)]
pub struct GitDiffResult {
    #[serde(with = "uuid_str")]
    #[ts(type = "string")]
    pub worktree_id: Uuid,
    pub file_path: Option<String>,
    pub diff: String,
}
```

### TypeScript Definition (apps/web/src/types/protocol.ts)
```typescript
export interface GitDiffResult {
  type: 'GitDiffResult';
  worktreeId: string;
  filePath?: string;
  diff: string;
}
```

### Unified Schema
Fields:
- worktreeId: string (UUID as string)
- filePath?: string (optional, camelCase)
- diff: string

Note: The TypeScript interface includes `type: 'GitDiffResult'` as a discriminator, but the generated TS binding only includes the data fields. The discriminator is handled by the discriminated union type.
