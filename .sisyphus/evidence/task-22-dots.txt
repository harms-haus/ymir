Real-Time Update Mechanism:
---------------------------
1. WebSocket receives AgentStatusUpdate message
2. Zustand store updates agent session status
3. useAgentStatus hook detects change via useStore selector
4. React re-renders component with new status
5. StatusDot component updates color and animation
6. Update happens in < 100ms (imperceptible to user)

Status Dot States:
----------------
⚫ Idle - Gray, static
🟢✨ Working - Green, flashing (0.5s interval)
🟡💤 Waiting - Yellow, pulsing (2s interval)

Implementation Details:
----------------------
- Hook: useAgentStatus(worktreeId) → AgentStatusInfo | null
- Component: StatusDot({ status, size })
- Subscription: WebSocket AgentStatusUpdate & AgentOutput
- Update path: WebSocket → Zustand → useAgentStatus → StatusDot
- Performance: Minimal re-renders, stable selectors with useMemo
